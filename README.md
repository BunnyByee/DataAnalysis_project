## 1. 프로젝트 개요 
본 프로젝트는 **2014년부터 2022년까지의 공공 데이터를 활용하여, 2030대 비만율 변화와 생활습관·식습관 관련 지표 간의 상관관계**를 분석한 데이터 분석 프로젝트입니다.

최근 2030대를 중심으로 비만율이 증가하는 현상에 주목하여, 운동량, 식습관, 디저트 관련 관심도 등의 요인이 비만율 변화와 어떠한 관계를 가지는지 탐색하고자 하였습니다.

초기 기획 단계에서는 SNS API 데이터를 활용한 분석까지 고려하였으나, 데이터 접근성과 신뢰성을 고려하여 **공공 통계 데이터를 중심으로 분석 범위를 조정**했습니다.

## 4. 데이터 설명
###  4.1 분석 데이터 기간
- **2014년 ~ 2022년**을 기준으로 함
- 일부 연도(2014~2016)의 데이터가 존재하지 않는 경우, 평균값을 사용하여 시각화에 반영함
- 2022년 이후의 최신 데이터는 확보에 한계가 있었음


### 4.2 변수 구성 
### 1. 종속변수
- **비만율 (Obesity_rate)**
  - 출처: 서울특별시, 서울시 건강격차 모니터링

### 2. 독립변수 (생활습관·식습관 지표)
- **디저트 관련 검색 트렌드**
  -  출처: 네이버 데이터랩
  - 키워드 “디저트”
  - 연도별 검색량을 합산하여 추세 분석
   - Min-Max 스케일된 데이터이므로 절대값보다는 **증가 추세**에 초점을 둠
- **유산소 신체활동 실천율**
  - 출처: 국민건강영양조사 (질병관리청)
  - 중·고강도 신체활동 권고 기준을 충족한 인원의 비율
- **규칙적 체육활동 참여 여부 및 빈도**
  - 출처: 국민생활체육조사 (문화체육관광부)
  - 대한비만학회 권고 기준(주 5회 이상 운동)을 중심으로 분석
- **에너지·지방 과잉 섭취자 분율**
  - 출처: 국민건강영양조사
  - 에너지 적정비율을 초과하여 지방을 섭취한 인원의 비율
- **영양소별 1일 섭취량 기반 파생변수**
  - 출처: 국민건강영양조사
  -  설탕 섭취량과 총 칼로리를 이용하여 설탕의 kcal 비율 파생 변수 생성
  - 일부 연도의 결측치는 평균값으로 대체함


### 4.3 분석에서 제외한 데이터

- **연간 음주율**
  - ‘최근 1년간 1회 이상 음주’라는 정의가 포괄적
  - 연도별 변화가 뚜렷하지 않아 비만율과의 관계 해석이 어려움
- **음주량 데이터**
  - 시기별 변화 폭이 크지 않아 설명력이 낮다고 판단하여 제외


## 5. 분석 과정
1. 연도별 데이터 수집 및 전처리
2. 변수별 시각화를 통한 추세 확인
3. 비만율과 독립변수 간 상관관계 분석
4. 다중 회귀분석을 통한 변수 간 관계 탐색
5. 변수 선택 기법을 활용한 주요 영향 변수 도출


## 6. 분석 방법
- **상관관계 분석**
  - 변수 간 양의 상관관계 / 음의 상관관계를 시각적으로 확인
- **회귀분석**
  - 비만율을 종속변수로 설정
  - 생활습관·식습관 관련 지표를 독립변수로 설정
  - 전진선택법, 후진제거법, 단계적 선택법을 적용하여 유의한 변수를 탐색함
  - 세 가지 변수 선택 기법 모두 동일한 결과를 도출함


## 7. 분석 결과
- **규칙적 체육활동**
  - 회귀계수(coef)가 음수
  - 비만율과 **반비례 관계**를 보임
- **디저트 관련 검색 트렌드**
  - 회귀계수(coef)가 양수
  - 비만율과 **비례 관계**를 보임
- 선택된 변수들의 p-value는 유의수준 이하로, 통계적으로 유의미한 관계를 가지는 것으로 확인됨


## 8. 결론
공공 데이터 기반의 시각화 및 회귀분석 결과, **2030대 비만율은 규칙적인 체육 활동과는 음의 관계, 디저트 관련 소비·관심 지표와는 양의 관계**를 가지는 경향을 확인했습니다.

본 분석은 인과관계를 단정하기보다는, 비만율 변화와 함께 움직이는 생활·식습관 지표를 데이터 기반으로 탐색하는 데 목적이 있습니다.

## 9. 한계점 및 개선 방향

- 지역·연령 세분화 데이터가 제한적임

향후 SNS 데이터 및 추가 연도 데이터를 확보한다면 분석의 범위와 정확도를 확장할 수 있습니다.
